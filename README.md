# BuildLab

BuildLab is a full-stack MVP application for creating and organizing game builds, team setups and strategy notes across different games.

The project was created to practice a basic full-stack workflow using Angular, TypeScript, C# and .NET Web API.

## Tech Stack

- Angular
- TypeScript
- C#
- .NET Web API
- Git

## Features

- Create build and strategy plans
- Display all saved builds
- Filter builds by game and status
- Update build status
- Delete build plans
- View build details
